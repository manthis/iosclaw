# Screenshots

This folder contains screenshots of iOSclaw for documentation.

## Taking Screenshots

### iOS Simulator

1. Start the app: `npx expo start` then press `i`
2. In Simulator, press `Cmd + S` to save a screenshot
3. Rename and move to this folder

### Expected Screenshots

| File | Screen | Description |
|------|--------|-------------|
| `connection-screen.png` | ConnectionScreen | Login screen with Gateway URL and token fields |
| `chat-empty.png` | ChatScreen | Empty chat with HAL 9000 greeting |
| `chat-conversation.png` | ChatScreen | Active conversation with messages |
| `chat-typing.png` | ChatScreen | Typing indicator while AI is responding |
